Live Price Sheet (Google Sheets)

Live prices for US, TASE, Europe & FX come from a Google Sheet using GOOGLEFINANCE. One-time setup:

  1. Create a new Google Sheet.
  2. Copy the formulas below and paste into cell A1.
  3. File ▸ Share ▸ Publish to web ▸ choose this sheet ▸ Comma-separated values (.csv) ▸ Publish.
  4. Paste the published .csv link into the URL field below and Save.
TASE tickers that show #N/A need the correct TLV symbol in a holding's "Sheet symbol" field.
